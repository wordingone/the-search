# Security Policy

## Scope

FoldCore is a research prototype. It is not designed for production deployment or processing of sensitive data.

## Reporting a Vulnerability

If you discover a security issue, please report it by opening a GitHub issue. This project does not process user data or run as a service, so the risk surface is limited to code execution in local research environments.

## Supported Versions

Only the latest commit on `master` is supported.
